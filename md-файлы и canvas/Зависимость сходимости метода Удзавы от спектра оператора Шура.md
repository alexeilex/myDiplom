Пусть после исключения скорости из [[Стандартная дискретная форма задачи Стокса|дискретной формы задачи Стокса]] получено уравнение на давление
$S p = g, \qquad S = BA^{-1}B^T,$ где \(S\) — [[оператор Шура]].

Классический [[метод Узавы]] для давления имеет вид
$p^{k+1} = p^k + \rho\,(g - Sp^k),$
где $(\rho > 0)$— параметр релаксации.

### Уравнение для ошибки

Обозначим ошибку
$e^k = p^k - p^\ast$,
где $(p^\ast)$— точное решение. Тогда, так как $(Sp^\ast = g)$, получаем
$e^{k+1} = (I - \rho S)e^k$.
Следовательно, сходимость метода определяется спектром матрицы \(S\).
### Условие сходимости

Если $(\lambda_i)$ — собственные значения (S), то для каждой моды ошибки имеем
$e_i^{k+1} = (1 - \rho \lambda_i)e_i^k.$
Чтобы метод сходился, необходимо и достаточно, чтобы
$|1 - \rho \lambda_i| < 1 \quad \text{для всех } i$.
Отсюда следует условие
$0 < \rho < \frac{2}{\lambda_{\max}},$
где $(\lambda_{\max})$ — наибольшее собственное значение \(S\).

### Скорость сходимости

Скорость уменьшения ошибки задаётся величиной
$\max_i |1 - \rho \lambda_i|.$
Чем ближе все собственные значения \(S\) к одному и тому же числу, тем быстрее сходится метод.
Если спектр \(S\) сильно растянут, то метод сходится медленно.
### Оптимальный шаг
Для симметричного положительно определённого \(S\) оптимальный параметр релаксации равен

$\rho_{\mathrm{opt}} = \frac{2}{\lambda_{\min} + \lambda_{\max}}.$
Тогда коэффициент сжатия ошибки становится
$\frac{\lambda_{\max} - \lambda_{\min}}{\lambda_{\max} + \lambda_{\min}}.$
### Вывод

Метод Удзавы является по сути методом Ричардсона для оператора Шура.  
Поэтому его сходимость полностью определяется спектром \(S\):

- если спектр узкий и хорошо сгруппирован, метод сходится быстро;
- если спектр растянут, метод сходится медленно;
- если удаётся подобрать [[предобуславливатель]] $(\tilde S)$ так, что $(\tilde S^{-1}S \approx I)$, то сходимость резко ускоряется.
