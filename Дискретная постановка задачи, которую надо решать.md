Да. Давай соберём это как **полную дискретную модель** для кода на **разнесённой сетке**: давление в центрах ячеек, скорости на гранях. Это как раз тот случай, который тебе нужен.

---

## 1) Область и сетка

Берём для старта прямоугольную область

[  
\Omega = [0,L_x]\times[0,L_y].  
]

Разбиваем её на равномерную сетку:

[  
h_x=\frac{L_x}{N_x},\qquad h_y=\frac{L_y}{N_y}.  
]

### Где лежат неизвестные

- давление (p_{i,j}) — в центре ячейки:  
    [  
    i=1,\dots,N_x,\quad j=1,\dots,N_y  
    ]
    
- горизонтальная скорость (u_{i,j}) — на вертикальных гранях:  
    [  
    i=0,\dots,N_x,\quad j=1,\dots,N_y  
    ]
    
- вертикальная скорость (v_{i,j}) — на горизонтальных гранях:  
    [  
    i=1,\dots,N_x,\quad j=0,\dots,N_y  
    ]
    

Это и есть MAC-сетка.

---

## 2) Непрерывная задача

Стационарные уравнения Стокса:

[  
-\nu \Delta \mathbf{u} + \nabla p = \mathbf{f},  
\qquad  
\nabla\cdot \mathbf{u}=0.  
]

В 2D:

[  
\mathbf{u}=(u,v),\qquad \mathbf{f}=(f_x,f_y).  
]

---

## 3) Граничные условия

Для твоей физической картины минимально нужно следующее.

### Стенки сверху и снизу

Если это твёрдые стенки, то обычно берут no-slip:

[  
u=0,\qquad v=0 \quad \text{на стенках}.  
]

### Слева и справа

Если ты задаёшь входное давление, то надо закрыть задачу и на выходе. Самый простой вариант для диплома:

[  
p = P_{\text{in}} \quad \text{слева},\qquad  
p = P_{\text{out}} \quad \text{справа}.  
]

Если задано только (P_{\text{in}}), то задача ещё не замкнута: нужно либо

- задать (P_{\text{out}}),
    
- либо зафиксировать среднее давление,
    
- либо задать другой тип выхода.
    

### Почему это важно

Давление в Стоксе определено с точностью до константы.  
Поэтому одна заданная граница давления — ещё не всегда полная постановка.

---

## 4) Дискретный оператор дивергенции

В каждой ячейке ((i,j)) условие несжимаемости записывается как

[  
\frac{u_{i,j}-u_{i-1,j}}{h_x}  
+  
\frac{v_{i,j}-v_{i,j-1}}{h_y}  
=0.  
]

Это и есть дискретная форма

[  
\nabla\cdot \mathbf{u}=0.  
]

Здесь:

- (u_{i,j}) — скорость через правую вертикальную грань ячейки,
    
- (u_{i-1,j}) — скорость через левую вертикальную грань,
    
- (v_{i,j}) — скорость через верхнюю горизонтальную грань,
    
- (v_{i,j-1}) — скорость через нижнюю горизонтальную грань.
    

---

## 5) Дискретный градиент давления

### Для уравнения по (u)

На вертикальной грани между ячейками ((i-1,j)) и ((i,j)):

[  
(\partial_x p)_{i,j}\approx \frac{p_{i,j}-p_{i-1,j}}{h_x}.  
]

### Для уравнения по (v)

На горизонтальной грани между ячейками ((i,j-1)) и ((i,j)):

[  
(\partial_y p)_{i,j}\approx \frac{p_{i,j}-p_{i,j-1}}{h_y}.  
]

---

## 6) Дискретный лапласиан для скоростей

Для горизонтальной скорости (u) на внутренних вертикальных гранях:

# [  
-\nu \Delta_h u_{i,j}

-\nu\left(  
\frac{u_{i+1,j}-2u_{i,j}+u_{i-1,j}}{h_x^2}  
+  
\frac{u_{i,j+1}-2u_{i,j}+u_{i,j-1}}{h_y^2}  
\right).  
]

Для вертикальной скорости (v):

# [  
-\nu \Delta_h v_{i,j}

-\nu\left(  
\frac{v_{i+1,j}-2v_{i,j}+v_{i-1,j}}{h_x^2}  
+  
\frac{v_{i,j+1}-2v_{i,j}+v_{i,j-1}}{h_y^2}  
\right).  
]

---

## 7) Полная дискретная система

### Уравнение для (u)

Для внутренних вертикальных граней:

# [  
-\nu \Delta_h u_{i,j}  
+  
\frac{p_{i,j}-p_{i-1,j}}{h_x}

f^x_{i,j}.  
]

### Уравнение для (v)

Для внутренних горизонтальных граней:

# [  
-\nu \Delta_h v_{i,j}  
+  
\frac{p_{i,j}-p_{i,j-1}}{h_y}

f^y_{i,j}.  
]

### Условие несжимаемости в каждой ячейке

[  
\frac{u_{i,j}-u_{i-1,j}}{h_x}  
+  
\frac{v_{i,j}-v_{i,j-1}}{h_y}  
=0.  
]

---

## 8) Матрицы

Если собрать всё в один вектор неизвестных

[  
U = \begin{pmatrix} u \ v \end{pmatrix},  
\qquad  
P = p,  
]

то получаем блочную систему

# [  
\begin{pmatrix}  
A & B^T\  
B & 0  
\end{pmatrix}  
\begin{pmatrix}  
U\ P  
\end{pmatrix}

\begin{pmatrix}  
F\ 0  
\end{pmatrix}.  
]

Где:

- (A) — дискретный оператор вязкости, по сути блок диагональный лапласиан для (u) и (v);
    
- (B) — дискретная дивергенция;
    
- (B^T) — дискретный градиент давления.
    

Если вязкость постоянна и схема стандартная, то

[  
A \approx \nu  
\begin{pmatrix}  
L & 0\  
0 & L  
\end{pmatrix},  
]

где (L) — дискретный лапласиан.

---

## 9) Как это связано с Узавой

На шаге Uzawa обычно делают так:

[  
A U^{k+1} = F - B^T P^k,  
]

[  
P^{k+1} = P^k + \omega, B U^{k+1}.  
]

То есть:

1. берёшь текущее давление,
    
2. решаешь уравнение на скорость,
    
3. считаешь нарушение несжимаемости,
    
4. обновляешь давление.
    

И вот здесь ключевое место:

[  
A U^{k+1} = F - B^T P^k  
]

для твоей схемы означает решение **двух дискретных уравнений Пуассона** — по одному на каждую компоненту скорости.

---

## 10) Что нужно для кода

Для рабочего минимального солвера тебе нужны:

1. массивы  
    [  
    u[N_x+1, N_y],\quad v[N_x, N_y+1],\quad p[N_x, N_y]  
    ]
    
2. функция для дискретного лапласиана на faces;
    
3. функция для градиента давления на faces;
    
4. функция для дивергенции в cells;
    
5. шаг решения уравнения на скорость;
    
6. обновление давления.
    

---

## 11) Как это формулировать в дипломе

Можно писать так:

> Для численного решения стационарной задачи Стокса используется разнесённая конечно-разностная схема (MAC-grid), в которой давление аппроксимируется в центрах ячеек, а компоненты скорости — на гранях. Дискретизация приводит к блочной системе с матрицами вязкости, дивергенции и градиента, после чего метод Узавы сводится к последовательному решению дискретных эллиптических задач для компонент скорости и корректировке давления по невязке несжимаемости.
