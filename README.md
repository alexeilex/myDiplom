В данной работе рассматривается стационарная задача Стокса, описывающая медленное вязкое течение несжимаемой жидкости в области сложной геометрии.

Задача формулируется в виде:

- уравнения равновесия (баланса импульса),
- условия несжимаемости.

Для численного решения используется метод конечных элементов / конечных разностей с раздельной аппроксимацией давления и скорости (например, на разнесённых сетках).

## Численный метод

Решение строится с использованием метода Узавы, который сводит исходную седловую систему к последовательности более простых подзадач:

решения уравнений типа Пуассона для компонент скорости,
обновления давления через оператор Шура.

Таким образом, одна итерация метода включает решение нескольких эллиптических задач вместо прямого решения полной системы.

## Цель работы
Основная цель исследования — изучить влияние:
- выбора аппроксимации операторов (в частности, лапласиана),
- разнесения сеток для давления и скорости,
- граничных условий,
на сходимость метода Узавы.

Особое внимание уделяется вопросу: **Можно ли подобрать такую дискретизацию и постановку задачи, при которой метод Узавы сходится за одну итерацию?**

## Идея исследования

Метод Узавы можно интерпретировать как итерационный процесс для уравнения на давление с оператором Шура. Теоретически, при "идеальном" согласовании дискретных операторов возможно сведение итерационного процесса к точному решению за один шаг.

В работе исследуется, достижимо ли это на практике для рассматриваемой геометрии области:

- область течения — односвязная,
- границы сверху и снизу — твёрдые стенки,
- давление задаётся на входе и выходе.

## Практическая значимость

Результаты могут позволить дедам решать Узаву не на суперкомпьютере, а проще. Еще можно существенно ускорить решение задач Стокса, лучше понять структуру дискретных операторов, может построить эффективные предобусловливатели для седловых систем.
