
[[Стандартная дискретная форма задачи Стокса]]:
Обычно для несжимаемого случая (g=0), но для вывода удобнее оставить общий вид.
Из первого уравнения:

$A u + B^T p = f$$\quad\Rightarrow\quad$$u = A^{-1}(f - B^T p).$
Подставляем во второе: $B A^{-1}(f - B^T p) = g$.  
Раскрываем:$BA^{-1}f - BA^{-1}B^T p = g$.  
Вводим [[оператор Шура]]: $S := BA^{-1}B^T$.  
Тогда получаем уравнение только на давление:
$S p = BA^{-1}f - g.$  
Обозначим правую часть как $d := BA^{-1}f - g$.  
Тогда: $S p = d$.

## 3) Классический Uzawa как Ричардсон

Итерация Узавы по давлению в этом виде записывается как
$p^{k+1} = p^k + \omega (d - S p^k).$
Это буквально итерация Ричардсона для системы (Sp=d).

---


Пусть $(p^\ast)$ — точное решение:
$S p^\ast = d$
Определим ошибку:
e_p^k := p^k - p^\ast.  
Тогда
$e_p^{k+1}  = p^{k+1} - p^\ast  = p^k + \omega(d - S p^k) - p^\ast.$  
Теперь используем $(d = S p^\ast)$:
$e_p^{k+1}  = p^k - p^\ast + \omega(S p^\ast - S p^k).$  
Вынесем разность:
$e_p^{k+1}  = e_p^k - \omega S e_p^k= (I - \omega S)e_p^k.$  
Вот и получилась формула:
$\boxed{e_p^{k+1}=(I-\omega S)e_p^k.}$  
---

## 5) Что из этого сразу следует

Итерация сходится тогда и только тогда, когда оператор
$I-\omega S$
является сжимающим, то есть его [[Спектральный радиус матрицы]] меньше 1:
$\rho(I-\omega S) < 1.$
Если (S) симметричен и положительно определён, то это сводится к условию на собственные значения:
$|1-\omega \lambda_i|<1 \quad \text{для всех } i.$
Отсюда: $0<\omega<\frac{2}{\lambda_{\max}}$  

---

## 6) Предобуславленный случай

Если вместо точного (S) ты используешь аппроксимацию ($\tilde S$), то итерация обычно пишется так:
$p^{k+1} = p^k + \omega \tilde S^{-1}(d - S p^k).$ 
Тогда
$e_p^{k+1}= p^{k+1} - p^\ast= p^k - p^\ast + \omega \tilde S^{-1}(d - S p^k).$  
Снова подставляем $(d = S p^\ast)$:
$e_p^{k+1}  = e_p^k + \omega \tilde S^{-1}(S p^\ast - S p^k).$
$e_p^{k+1}  = e_p^k - \omega \tilde S^{-1}S e_p^k$
Итак:
$\boxed{e_p^{k+1} = (I - \omega \tilde S^{-1}S)e_p^k.}$

---

## 7) Что здесь важно понимать

Это уже не просто красивая формула, а ключ к анализу:

- если $(\tilde S = S)$, то получаем точный Richardson по Шуру;
    
- если $(\tilde S \approx S)$, то спектр $(\tilde S^{-1}S)$ должен быть узким;
    
- если $(\tilde S^{-1}S \approx I)$, то метод почти схлопывается за один шаг.
    

Можно написать так:

> После исключения скорости из дискретной системы Стокса получается уравнение на давление ($S p = d$), где ($S = BA^{-1}B^T$) — оператор Шура. Метод Удзавы для этого уравнения эквивалентен методу Ричардсона, поэтому ошибка давления удовлетворяет рекуррентному соотношению  
> $e_p^{k+1} = (I - \omega S)e_p^k.$
> В предобусловленном варианте с аппроксимацией (\tilde S) получаем  
>$e_p^{k+1} = (I - \omega \tilde S^{-1}S)e_p^k.$
---

По идее из этого можно получить
1. условие сходимости,
    
2. оптимальный шаг (\omega),
    
3. оценку числа итераций через спектральное число обусловленности.
